# Leftover-Locator

Leftover Locator is a full-stack web application designed to help users find restaurants offering discounted "surprise bags" of leftover food. The app features an interactive map, built with the Google Maps API, displaying restaurant locations along with details about available bags, pricing, and food types.

# Key Features:

- Interactive Map: Displays restaurant locations with dynamic markers that show bag availability and pricing.
- Data Collection: Uses Google Forms and the Google Sheets API to collect and store restaurant and reservation data in a PostgreSQL database.
- Responsive Web Application: Built with React for the frontend, ensuring a seamless experience on both desktop and mobile devices.
- Backend: Implemented a RESTful backend using Node.js to handle data requests and interactions between the frontend and database.
